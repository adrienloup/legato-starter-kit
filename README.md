# Legato Starter Kit
A starter boilerplate using Webpack 4 (Legato codename) for basic web development needs.

## Build Setup
```
# install dependencies
npm install

# run boilerplate
npm start

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
